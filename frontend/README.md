# Frontend (React + Vite)

## Run

```bash
npm install
cp .env.example .env
npm run dev
```

Default local URL: `http://127.0.0.1:5173`

## Environment

- `VITE_BACKEND_URL` (default: `http://127.0.0.1:5000/api/disaster`)

## UI Sections

- Simulation Input Form (location, scenario mode, population, blocked roads)
- Situation Report (risk + hazards + weather)
- Resource Allocation
- Evacuation Routes
- Agent Reasoning timeline
- Simulation History
